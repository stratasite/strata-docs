# Strata Developer Documentation

Official documentation for Strata Semantic Analytics - the semantic layer platform for building intelligent data applications.

## About This Repository

This repository hosts the official developer documentation for Strata. The documentation is publicly accessible to help data engineers and developers build semantic models, integrate with the Strata API, and leverage semantic analytics capabilities.

## Documentation Site

Visit the live documentation at: **https://strata.do/developer-docs/**

## What You'll Find

- **Getting Started Guides** - Installation, setup, and core concepts
- **CLI Reference** - Command-line tools for managing semantic models
- **Semantic Model Documentation** - Building tables, relationships, and dimensions
- **API Integration** - REST API and LLM integration guides
- **Advanced Topics** - Semantic routing, universe formation, and optimization
- **Examples & Recipes** - Real-world use cases and patterns

## About Strata

Strata is a semantic analytics platform that enables data engineers to build semantic models and expose them through natural language interfaces and APIs. Learn more at [strata.do](https://strata.do).
