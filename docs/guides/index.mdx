## Learning Path

If you're new to Strata, follow this learning path:

1. Start with [Creating Tables](/docs/guides/creating-tables) to understand the basics
2. Learn about [Fields and Types](/docs/guides/fields-and-types) to define your data correctly
3. Set up [Relationships](/docs/guides/relationships) to connect your tables
4. Configure [Datasources](/docs/semantic-model/datasources) for your databases
5. Learn about [Deployment](/docs/guides/deployment) to get your models into production
