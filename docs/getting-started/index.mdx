# Getting Started

Welcome to Strata! This guide will help you get started with building semantic analytics models using the Strata platform.

## Your Journey

To begin working with Strata, you'll need to:

1. **Learn about Strata CLI** - Understand what the CLI is and how it helps you build semantic models
2. **Install Strata CLI** - Install the command-line tool on your system
3. **Create your first project** - Initialize a new Strata project and start building

Let's start by learning about [Strata CLI](/docs/getting-started/strata-cli), which is the foundation for everything you'll do with Strata.
