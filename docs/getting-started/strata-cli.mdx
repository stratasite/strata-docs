# Strata CLI

Strata CLI is a command-line interface tool that helps you create, manage, and deploy Strata Semantic Analytics projects. It's your primary tool for working with the Strata platform from your terminal.

## What is Strata CLI?

Strata CLI provides a comprehensive set of commands for managing your Strata projects. You can use it to configure datasource connections, define semantic models with AI assistance, run audits and validations, and deploy projects to Strata servers.

## What you can do with Strata CLI

- **Manage datasource connections** - Configure and test connections to various data warehouses including PostgreSQL, MySQL, SQL Server, Snowflake, Athena, Trino, and more
- **Create semantic table models** - Build semantic models from datasource tables with AI-powered field suggestions
- **Run audits and validations** - Validate YAML syntax, model definitions, and datasource connections
- **Deploy projects** - Deploy your semantic models to Strata servers for production use
