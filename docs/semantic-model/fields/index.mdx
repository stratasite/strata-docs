# Fields

Fields are the building blocks of your semantic model: dimensions, measures, data types, and formatters.

## Overview

- **[Dimensions](/semantic-model/fields/dimensions)** — Categorical attributes for grouping and filtering
- **[Measures](/semantic-model/fields/measures)** — Aggregatable metrics (sum, count, avg, etc.)
- **[Data Types](/semantic-model/fields/data-types)** — `string`, `integer`, `decimal`, `date`, `date_time`, `boolean`, and more
- **[Formatters](/semantic-model/fields/formatters)** — Currency, percent, and custom value formatting

## Quick Reference

| Field type | Use in GROUP BY | Aggregation | Example |
|------------|-----------------|-------------|---------|
| Dimension  | Yes             | No          | Customer Name, Region, Order Date |
| Measure    | No              | Yes (required) | Total Revenue, Order Count |
