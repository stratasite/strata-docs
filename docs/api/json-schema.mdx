# JSON Schema

JSON Schema definitions for all Strata YAML file types.

## Overview

JSON Schema provides machine-readable definitions of all YAML file structures used in Strata projects.

## Available Schemas

### Table Schema

```json
{
  "$schema": "http://json-schema.org/draft-07/schema#",
  "type": "object",
  "required": ["datasource", "name", "physical_name", "cost", "fields"],
  "properties": {
    "datasource": {"type": "string"},
    "name": {"type": "string"},
    "physical_name": {"type": "string"},
    "cost": {"type": "integer"},
    "fields": {
      "type": "array",
      "items": {"$ref": "#/definitions/field"}
    }
  }
}
```

### Relation Schema

```json
{
  "$schema": "http://json-schema.org/draft-07/schema#",
  "type": "object",
  "required": ["datasource"],
  "properties": {
    "datasource": {"type": "string"},
    "relationships": {
      "type": "object",
      "patternProperties": {
        ".*": {
          "type": "object",
          "required": ["left", "right", "sql", "cardinality"],
          "properties": {
            "left": {"type": "string"},
            "right": {"type": "string"},
            "sql": {"type": "string"},
            "cardinality": {
              "enum": ["one_to_one", "one_to_many", "many_to_one"]
            }
          }
        }
      }
    }
  }
}
```
